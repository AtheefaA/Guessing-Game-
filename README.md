# Guessing Game (Python)

This project is a simple Python guessing game where the computer
generates a random number and the user tries to guess it.

## Features
- Random number generation
- User input handling
- Hint messages (Too High / Too Low)
- Beginner-friendly Python code

## Technologies Used
- Python
- Random module

## How It Works
The program generates a random number within a range.
The user enters guesses until the correct number is found,
with hints provided after each attempt.

## Example
Guess the number: 5  
Too low! Try again.  
Guess the number: 8  
Correct! You guessed it.
